pgxl 是一个基于 pg 可扩展的 架构，安装部署在几台机器上

postgresql 有兴趣的可以了解一下，也是一种关系型数据库。postgresql 配合 pg-xl 组件，可以达到类似 hadoop 一样的分布式效果。而且还有分布式事务的特性哦
