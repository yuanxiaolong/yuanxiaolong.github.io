介绍一下Hbase单机伪分布式安装

hbase是一个列式数据库，基于hdfs文件的。跟mongodb同属nosql系列，具有海量insert和key-value简单高效查询的作用。

环境准备

  • hadoop环境，伪分布式或集群环境
  • hbase 适当版本对应hadoop-1.x、hadoop-2.x 的安装包，可以在这里下载
