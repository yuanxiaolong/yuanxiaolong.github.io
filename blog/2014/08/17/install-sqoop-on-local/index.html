介绍一下单机安装sqoop的过程

sqoop（sql to hadoop），用于数据交换。即可以从关系型DB如mysql，导入到hdfs、hive、hbase。当然也有导出功能。（本文只示例导入hdfs、hbase，导出等只是参数不同而已）

环境准备

  • hadoop（1.2.1）
  • hive （0.13.1）
  • hbase （0.98.4-hadoop1）
  • mysql （5.6.16）
注：括号里的是我本机的实验环境，并非必要条件
